Bienvenido al Panel

Tenemos cursos orientados a la capacitación tecnológica.

Cursos Ofrecidos

Diseño Web

Aprende a crear sitios web modernos y responsivos utilizando las últimas tecnologías como HTML, CSS y JavaScript. Este curso está diseñado para principiantes y profesionales que desean mejorar sus habilidades en diseño web.

Programación

Domina los fundamentos de la programación y desarrolla aplicaciones utilizando lenguajes como Python, Java y más. Este curso incluye proyectos prácticos para que puedas aplicar tus conocimientos en situaciones reales.

Diseño Gráfico

Explora el mundo del diseño visual y aprende a crear gráficos impactantes utilizando herramientas como Adobe Photoshop e Illustrator. Este curso te ayudará a desarrollar un portafolio profesional para destacar en el mercado laboral.
