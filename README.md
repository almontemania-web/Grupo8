🎓 Sistema de Registro de Estudiantes - Grupo 8
Este es un panel universitario interactivo diseñado para la gestión y registro de estudiantes en diversos cursos técnicos. El proyecto se enfoca en ofrecer una interfaz limpia, profesional y responsiva, utilizando una paleta de colores institucional.

🎛️ Características
Panel de Control (Dashboard): Visualización de bienvenida y catálogo de cursos ofrecidos.

Formulario de Registro: Sistema de captura de datos que incluye validación de nombre, matrícula, correo, fecha de nacimiento y selección de curso.

Diseño Responsivo: Adaptabilidad para dispositivos móviles y tablets

Navegación Intuitiva: Menú centralizado para alternar fácilmente entre el inicio y el registro.

🛠️ Tecnologías Utilizadas
HTML5: Estructura semántica del contenido.
CSS3: Estilo personalizado utilizando variables para colores institucionales (--primary-color: #0b1838 & (--secondary-color: #F68121;)) y Flexbox para el diseño.

🎨 Interfaz y Diseño
El proyecto utiliza una combinación de colores del campus virtual:

Azul Primario (#0b1838): Utilizado en encabezados y pies de página para transmitir seriedad.

Naranja Secundario (#F68121): Utilizado en la barra de navegación para resaltar la interactividad.

Sección de Cursos: Cada artículo cuenta con sombreado dinámico (box-shadow) y bordes redondeados para un aspecto moderno.
